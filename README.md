# Webdriver Fiddle

## WIP

Webdriver fiddle built with Svelte, xterm, ace editor

The backend portion of the fiddle is [here](https://github.com/WillBrock/webdriver-fiddle-server)

## Install and start
```bash
npm install
npm run dev
```

## Development url once started
```bash
http://localhost:5000
```
